# README.md

## Features

- 🕒 Real-time updating digital clock
- 📅 Displays current date and weekday
- 🖥️ Responsive layout adapts to terminal size
- 🚦 Graceful exit handling

## Installation

```bash
pip install clock-app
```

## Usage

Simply run:

```bash
clock
```

## Requirements

- Python 3.6+
- Rich library
